## Overview

The Yoga Class Planner skill helps yoga teachers create complete, well-structured 60-minute class sequences with detailed pose instructions, timing, transitions, and teaching cues.

## What This Skill Does

- Generates balanced 60-minute beginner-friendly sequences
- Provides detailed pose-by-pose instructions with timing
- Creates smooth transitions between poses
- Includes warm-ups, peak poses, and cool-downs
- Offers teaching cues and modifications for beginners
- Structures classes with proper pacing and energy flow

## Class Structure

Every generated class includes:
- **Opening/Centering** (5 minutes)
- **Warm-up Sequence** (10-12 minutes)
- **Standing Poses** (15-18 minutes)
- **Peak Pose Section** (10-12 minutes)
- **Cool-down/Floor Poses** (12-15 minutes)
- **Savasana & Closing** (5-8 minutes)
