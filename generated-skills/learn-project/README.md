# Learn Project Skill (Codebase Analyzer)

> Generate comprehensive, developer-focused codebase analysis using modular templates

## Quick Start

```
"Analyze the codebase in /path/to/project"
"Generate codebase_analysis.md for this project"
"Help me understand this codebase structure"
```

## Overview

The Learn Project skill automatically detects your project's technology stack and generates detailed codebase analysis by combining modular templates specific to your technologies.

## What This Skill Does

- Automatically detects project technologies (TypeScript, React, MCP, Cloudflare Workers, iOS, etc.)
- Loads appropriate analysis templates for detected tech stack
- Generates comprehensive `codebase_analysis.md` document
- Provides file-by-file breakdowns
- Includes architecture diagrams and insights
- Offers technology-specific recommendations

## Supported Technologies

- **MCP Server** - Model Context Protocol servers
- **TypeScript** - TypeScript projects
- **Cloudflare Workers** - Edge computing applications
- **Jest Testing** - Jest test suites
- **iOS Swift** - iOS applications
- **Android Kotlin** - Android applications
- **Next.js** - Next.js applications
- **React** - React applications
- **Python** - Python projects
