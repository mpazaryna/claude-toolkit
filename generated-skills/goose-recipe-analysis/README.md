## Overview

The Goose Recipe Analysis skill helps you create Goose recipes specifically for document analysis and transformation - working with saved data rather than querying live sources.

## What This Skill Does

- Create recipes for analyzing saved reports (markdown, JSON, CSV)
- Transform or enrich existing documents
- Extract insights and generate recommendations
- Create shell script runners for easy execution
- Generate recipes from specification files

## When to Use

**Use this skill for:**
- Analyzing previously generated reports
- Follow-up analysis on saved data
- Document transformation and enrichment
- Extracting actionable insights from reports

**Use `goose-recipes` skill instead for:**
- Querying live data sources
- Recipes requiring MCP server access
- Structured JSON validation needs
