## Troubleshooting

### Skill Not Recognized

**Solutions:**
1. Verify: `ls ~/.claude/skills/goose-recipe-analysis/SKILL.md`
2. Check YAML frontmatter is valid
3. Try explicit: "Use goose-recipe-analysis skill to..."
4. Restart Claude Code

### Templates Not Loading

**Solutions:**
1. Verify: `ls ~/.claude/skills/goose-recipe-analysis/assets/`
2. Check file names match exactly:
   - `analysis-recipe-template.yaml`
   - `runner-script-template.sh`
3. Ensure read permissions: `chmod +r ~/.claude/skills/goose-recipe-analysis/assets/*`

### Goose Not Found

```bash
# Install Goose
pipx install goose-ai

# Verify
which goose
goose --version
```
