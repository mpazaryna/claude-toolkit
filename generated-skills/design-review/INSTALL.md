# Installing design-review

## Quick Install

```bash
cp -r design-review/ /path/to/your/project/.claude/skills/
```

## Verify Installation

```
.claude/
└── skills/
    └── design-review/
        ├── SKILL.md
        ├── README.md
        ├── HOW_TO_USE.md
        ├── INSTALL.md
        └── references/
            ├── app-store.md
            ├── accessibility-audit.md
            └── checklist.md
```

## Related Skills

- `design-principles` - Design theory
- `design-web` - Web implementation
- `design-swiftui` - SwiftUI implementation
