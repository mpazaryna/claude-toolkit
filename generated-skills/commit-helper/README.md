# Commit Helper Skill

> Generate clear, descriptive commit messages from git diffs

## Quick Start

```
"Generate a commit message for my staged changes"

```

## Overview

The Commit Helper skill analyzes your git staged changes and generates professional commit messages following best practices.

## What This Skill Does

When invoked, Claude will:
1. Run `git diff --staged` to see your changes
2. Analyze what was modified and why
3. Generate a commit message with:
   - Concise summary (under 50 characters)
   - Detailed description
   - List of affected components
