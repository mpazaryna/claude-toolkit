# Installing swift-ui

## Quick Install

```bash
cp -r swift-ui/ /path/to/your/project/.claude/skills/
```

## Global Install

For availability across all projects:

```bash
cp -r swift-ui/ ~/.claude/skills/
```

## Verify Installation

```
.claude/
└── skills/
```
