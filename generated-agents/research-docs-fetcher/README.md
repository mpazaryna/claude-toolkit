# Research Docs Fetcher Agent

> Systematically fetch, process, and organize web content into structured markdown files

## Quick Start

```
"Research the latest FastAPI documentation"
"Fetch documentation from https://docs.anthropic.com/claude/reference"
"Gather technical specs for TypeScript 5.0"
```

## Overview

The Research Docs Fetcher agent is a specialist for gathering documentation, technical specifications, and reference materials from the web, organizing them into structured markdown files in the `ai_docs/research` directory.

## What This Agent Does

- Fetches content from URLs or researches topics via web search
- Converts web content to clean, structured markdown
- Organizes research in `ai_docs/research/` directory
- Avoids re-fetching content updated within 24 hours
- Creates timestamped, well-organized documentation

## Core Capabilities

- **Direct URL Fetching** - Fetch specific documentation pages
- **Topic Research** - Search and gather materials on topics
- **Smart Caching** - Skip recently fetched content (24-hour window)
- **Markdown Conversion** - Clean, readable markdown output
- **Systematic Organization** - Structured file naming and storage

## When to Use

Use this agent when you need to:
- Gather API documentation for a new library
- Research technical specifications
- Collect reference materials for a project
- Fetch and organize multiple documentation pages
- Build a knowledge base from web sources

## Tools Available

- **WebFetch** - Fetch and process web content
- **Read** - Check existing research files
- **Glob** - Find existing documentation
- **Bash** - File operations and organization
