# Research Agent

> Systematic web research specialist for gathering and organizing documentation, technical specifications, and reference materials

## Quick Start

```
"Research the latest TypeScript best practices"
"Fetch documentation from https://example.com/api-docs"
"Gather information about React Server Components"
```

## Overview

The Research Agent systematically fetches, processes, and organizes web content into structured markdown files in the `ai_docs/research` directory, making it easy to build a knowledge base for your project.

## What This Agent Does

- Fetches content from URLs or searches for topics
- Checks for existing research to avoid duplication
- Processes HTML into clean, readable markdown
- Organizes content with metadata and timestamps
- Creates structured files in `ai_docs/research/`
- Provides summaries and key findings

## Core Principles

1. **Avoid Duplication** - Check existing files before fetching
2. **Structured Output** - Consistent markdown formatting with metadata
3. **Source Attribution** - Always include source URLs and timestamps
4. **Organized Storage** - Clear naming conventions and directory structure
5. **Actionable Summaries** - Key findings and follow-up suggestions

## When to Use

Use this agent when you need to:
- Research new technologies or frameworks
- Gather API documentation from multiple sources
- Build a reference library for your project
- Document technical specifications
- Create onboarding materials with external resources

## Output Format

- **Research Summary**: Overview of sources processed
- **Files Created/Updated**: List of generated markdown files
- **Key Findings**: Important information discovered
- **Follow-up Suggestions**: Related topics to explore
